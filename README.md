# ✦ Markdown 预览器

> 📝 **把 AI 聊天里的乱码，一键变成漂亮的文档。**

> 🧘 **Vibe Coding 作品** — 作者完全不懂编程，纯靠 AI 迭代开发。

---

## 🧩 这个工具是怎么来的

每次跟 AI 聊天，遇到写得好的内容，想复制下来保存成文档，结果贴到记事本或 Word 里全是乱码——表格乱了、公式变成一堆符号、代码缩进没了、列表挤成一团。

但是又不想为了存个文档去装一堆复杂的软件。

所以就做了这个工具：一个纯前端页面，把 AI 生成的 Markdown 内容重新排版成干净、美观的 HTML 文档。不需要安装任何东西，下载一个文件打开就能用。

---

## ✨ 用它来做什么

| 场景 | 怎么做 |
|------|--------|
| 保存 AI 对话记录 | 复制 AI 回复 → 粘贴到左侧 → 右侧自动排版 |
| 整理技术笔记 | 贴入 Markdown 源码 → 实时预览效果 |
| 分享带公式的内容 | 数学、物理公式自动渲染，不用截图 |
| 保存带流程图的内容 | Mermaid 图表自动变成图形 |
| 导出离线文档 | 一键导出独立 HTML 文件，发给谁都能打开 |

## ✅ 能做什么

- 实时预览：左边写/贴内容，右边即时看到排版效果
- 数学公式：行内公式 $E=mc^2$ 和块级公式都能正确渲染
- 代码高亮：Python、JavaScript 等多种语言有颜色区分
- 流程图渲染：写在 Markdown 里的流程图自动变成图形
- 任务列表：勾选框支持
- 亮色/暗色：手动切换，也跟随系统自动变化
- 导出 HTML：一键导出独立文件，内嵌所有样式，离线也能看
- 零安装：只有一个 .html 文件，双击用浏览器打开即可

## 💬 它是怎么来的

这个工具完全是用 AI 聊天做出来的。

作者不会写 HTML，不会 CSS，不会 JavaScript。只是每次跟 AI 说：

- *"帮我做个 Markdown 预览器，左右分栏的"*
- *"加一个暗色模式吧"*
- *"表格太长了能不能左右滑动？"*
- *"导出的时候能不能把样式打包进去，离线也能看"*

AI 生成代码 → 打开浏览器看一眼 → 不好用的地方再描述给 AI 修。就这么迭代出来的。

这就是 **Vibe Coding**：

> *"It's not really coding — you just describe what you want, the AI builds it, and you're amazed it works."* — Andrej Karpathy

## 🧰 技术栈（不用懂这些也能用）

- [markdown-it](https://github.com/markdown-it/markdown-it) — Markdown 解析
- [KaTeX](https://katex.org/) — LaTeX 公式渲染
- [Mermaid](https://mermaid.js.org/) — 图表渲染
- [Prism.js](https://prismjs.com/) — 代码高亮
- [github-markdown-css](https://github.com/sindresorhus/github-markdown-css) — GitHub 风格的样式

全部通过 CDN 加载，不需要安装任何东西。
